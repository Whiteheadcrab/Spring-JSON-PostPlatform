Spring-JSON-PostPlatform project is post online platform .
Main responsibility of this application :
* login
* manipulation of data inside already existing account (in implementation)
* creating new parcel for existing account (in implementation)
* manipulation of existing account (in implementation)
* generate Kod2d in a format byte table with base64 coding basing on data of existing account and selected parcel
